sfml.js

A cross-platform SFML's Node.js binding.

Installation

 
$ npm install --save sfml.js

Usage

Refer to Typing Definitions: https://github.com/XadillaX/node-sfml/blob/main/index.d.ts.

And SFML's official documentation: https://www.sfml-dev.org/documentation/2.5.1/.

Examples
  • node-sfml-demos: Several small example games;
  • play-nes: A cross-platform desktop NES emulator powered by OpenGL in Node.js.

PR is welcomed. You can add your own sfml.js example.
